1. Firstly, make sure you have istalled node.js in you machine
2. Open you IDE and initialize a Node.js project by running: npm init -y
3. Install the express package by running: npm install express in your machine
4. Make sure the index.js, index.html and style.css files are in the same directory
5. In index.js, open a terminal and navigate to the project directory.
6. Run the Node.js server: node index.js
7. Open a web browser and visit http://localhost:5000
8. You can test different languages by appending the language query parameter, for example: http://localhost:5000/index.html?language=French


9. This should be the project structure
project-folder/
├── index.js
├── index.html
├── styles.css

10. Feel free to ask if it return an error
